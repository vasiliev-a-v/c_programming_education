
Маленькие программы на Си, предназначенные для обучения программированию:
database    - пример создания простенькой базы данных
dictionary  - словарь иностранных слов
goroda      - игра в Города. Угадывать города
kassa       - программа в кассу
pole_chudes - игра "Поле чудес"
testing     - программа тестирования пользователя: список вопросов, в которых он должен выбирать правильные ответы. В конце ставится оценка.
